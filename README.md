# Array pairer

Generate a set of pairs from an array. One of the possible permutations of pairs will be returned.

## Installation

??

## Usage

```javascript
const createPairs = require("./array-pairer");

const pairs = createPairs(["a", "b", "c", "d"]);
console.log(pairs);
// [["b", "a"], ["a", "d"], ["d", "c"], ["c", "b"]];
```
